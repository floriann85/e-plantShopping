# e-plantShopping

The **e-plantShopping** project is a React application developed as part of the IBM Full Stack Application Developer Program.  
The application represents a small online shop for houseplants and demonstrates key frontend concepts such as:

- React components  
- React Hooks (useState, useEffect)  
- Redux Toolkit for state management  
- Dynamic rendering of product data  
- Event handling and conditional rendering  
- Navigation between multiple pages (Landing, Products, Cart)

## Project Overview

The application provides:

- A landing page with a start button  
- A navigation bar with links to *Paradise Nursery*, *Plants*, and the *Cart*  
- A product list with plant cards (image, name, description, price, Add‑to‑Cart button)  
- At least three plant categories  
- A shopping cart with quantity controls (+/−), total cost calculation, and a delete option  
- Buttons for *Continue Shopping* and *Checkout*
